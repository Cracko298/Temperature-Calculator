# Temperature-Calculator
A Simple Temperature Calculator. Supports Kelvin, Celsius, &amp; Fahrenheit Conversion.

# Features:
- Converts Kelvin, Celsius, & Fahrenheit.
- Rounds to The Nearest ***WHOLE NUMBER***.
